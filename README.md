# undergrad-thesis
# 機械学習による南米諸国の「資源の呪い」と制度的要因の因果分析

南米諸国（ADM2：地区レベル）を対象に、鉱山資源（鉱山の有無）が **地域の経済活動**（夜間光）および **紛争発生** に与える因果効果を推定し、さらに **制度の質（WGI）による効果の異質性** を検証する卒業研究プロジェクトです。

---

## 研究背景 / モチベーション

豊富な天然資源が経済成長を促すとは限らず、むしろ停滞や紛争を招く現象は「資源の呪い」として知られています。近年は、資源の影響は制度の質など文脈要因で左右されるという見方が強まっています。:contentReference[oaicite:2]{index=2}

代表的先行研究（Hodler et al., 2023）はサブサハラ・アフリカで制度の質が資源の呪いを緩和しうることを示唆しましたが、対象期間が 1997–2013 と比較的古く、また地域が限定されているため **時間的頑健性** と **外部妥当性** の検証が課題として残ります。

---

## 研究目的 / 新規性

本研究は、先行研究の知見の外部妥当性と時間的頑健性を検証するために

- 対象地域：南米諸国
- 対象期間：2013–2020
- 手法：因果フォレスト（Causal Forest DML）
- 目的：ATEだけでなく、制度の質（WGI）に応じたGATE（効果の異質性）を推定

を行います。

---

## データ概要（ADM2 / 2013–2020）

- **単位**：ADM2（地区）レベル  
- **処置（T）**：当該地区における **鉱山の有無**
- **アウトカム（Y）**：
  - 夜間光（経済活動の代理指標）
  - 紛争発生の有無  
- **共変量（X）**：気候・地形・地理的要因など（先行研究に準拠）
- **異質性変数（W）**：World Governance Indicators（WGI）に基づく制度の質指標 

---

## 推定手法：Causal Forest DML

因果効果推定には **Causal Forest DML** を用います。処置Tと結果Yを共変量Xで予測し、その残差を用いることで交絡の影響を調整し、残差に対して因果フォレストを適用して処置効果を推定します。これにより、非線形な交絡調整と、制度の質Wに依存するCATE推定を両立させます。

---

## 識別仮定（分析の前提）

本研究は、以下の仮定のもとで識別を行います

1. **条件付き独立**：共変量Xで条件付けると処置割当が潜在結果と独立
2. **共通支持**：処置群と対照群の傾向スコア分布が重複（極端なサンプル除外を検討）
3. **SUTVA（空間的波及への配慮）**：鉱山地区に隣接する地区も処置群に含め、波及効果を考慮
4. **外生性（ラグ処理）**：アウトカムに対して前年（t−1）の変数で説明するラグ設計を採用

---

## 結果（暫定）

### ATE（平均処置効果）
- 夜間光：**+1.9%**（95%CI: **[-5.6%, 9.4%]**）→ 0 を跨ぎ、有意差なし  
- 紛争：**-0.1%**（95%CI: **[-5.9%, 5.7%]**）→ 有意差なし

### GATE（制度の質による異質性）
制度の質が悪いグループ（Q1）ほど夜間光への効果が高いシグナルが見られる一方、各グループの信頼区間はいずれも 0 を跨ぎ、明確な異質性があると結論づけるのは困難でした。紛争についても同様に明確な異質性は確認されませんでした。

---

## 考察（暫定）

- 「制度が悪いほど効果が高い」シグナルは、貧困地域・制度未整備地域ほど鉱山開発の限界効果（インフラ・雇用）が大きい可能性を示唆する一方、頑健な証拠にはさらなるサンプル確保が必要。
- 先行研究との乖離要因として、地域・時代差（2010年代南米は制度・法整備が進み、劇的な変化が起きにくい可能性）が考えられます。

---

## 限界と今後の課題

本研究の主要な限界は、鉱山データの時系列性に制約がある点です。処置変数（鉱山の有無）は **2019年時点の衛星画像由来スナップショット** を用い、2013–2020年の全期間に適用して補完しているため、期間内の鉱山の新規稼働・操業停止などの動的変化を捉えきれていない可能性があります。より精緻な推定には、時系列変動を含む高品質な鉱山データの取得が課題です。
